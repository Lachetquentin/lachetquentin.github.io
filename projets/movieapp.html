Movie App

Informations

  • ○ Catégorie: Application mobile
  • ○ But: Projet personnel
  • ○ Date du projet: Août 2021

Application mobile en cours de développement afin d'apprendre le language React Native. J'utilise l'API de TheMovieDB afin d'accéder aux données sur les films.

Je compte ajouter une base de données afin de pouvoir gérer la connexion / inscription, et le stockage des films vu et mis en favoris etc... qui ne sont stocké pour l'instant que temporairement.

Les autres fonctionnalités prévu sont l'ajout d'une messagerie et d'un fil d'actualité.
